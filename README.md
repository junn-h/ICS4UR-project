# ICS4UR-project

## Overview
Dodge Blitz is a single-player game built with Java Swing. The player controls a spaceship at the bottom of the screen and must dodge falling objects while collecting power-ups. The game includes a login and registration system that tracks and saves each player's high scores to local files.

## What the Game Does
 
- Dodge four types of falling hazards: Normal, Fast, Bomb, and Zigzag, each with unique behavior
- Collect four types of power-ups: Coins (points), Shields (protection), Slow Time, and Extra Lives
- Progress through levels as the game speeds up over time
- Save scores per user and view a leaderboard of top players
- Register an account or play as a guest

## How to Run

1. Clone or download respository
2. Open project folder in an IDE
3. Set LoginFrame.java as the main class and run

## Goals

- Demonstrate object-oriented design using abstract classes and inheritance
- Implement collision detection between multiple moving game objects
- Build a functional GUI application using Java Swing
- Practice file I/O for persistent data storage
- Meaningfully extend an AI-generated starting point with well-organized additions
